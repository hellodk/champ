# Phase 2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement Memory Bank (cross-session fact persistence), Context Window Summariser (LLM compaction instead of drop-oldest), MCP Manager UI (server status panel with reload), and Agent Dashboard (state badges on session tabs).

**Architecture:** Each feature is independent. Memory Bank stores assistant responses as JSON facts at `.champ/memory.json` and injects recent facts into the system prompt. Context Summariser adds an async `fitWithSummary()` method that replaces dropped turns with an LLM summary. MCP Manager adds a popup panel to the action bar. Agent Dashboard adds state badges to existing session tabs.

**Tech Stack:** TypeScript, Node.js `fs/promises`, VS Code webview messaging, existing `AgentController`, `ContextWindowManager`, `McpRegistry`, `AgentManager` classes.

```typescript
import * as fs from "fs/promises";
import * as path from "path";

export interface MemoryItem {
  id: string;
  timestamp: number;
  userQuery: string;
  assistantSummary: string;
  sessionId: string;
}

const MAX_MEMORIES = 50;

export class MemoryBank {
  private items: MemoryItem[] = [];
  private loaded = false;
  private readonly filePath: string;

  constructor(workspaceRoot: string) {
    this.filePath = path.join(workspaceRoot, ".champ", "memory.json");
  }

  /** Load persisted memories from disk. Call once at startup. */
  async load(): Promise<void> {
    try {
      const raw = await fs.readFile(this.filePath, "utf-8");
      const parsed = JSON.parse(raw) as MemoryItem[];
      if (Array.isArray(parsed)) this.items = parsed;
    } catch (err: unknown) {
      if ((err as NodeJS.ErrnoException).code !== "ENOENT") {
        console.warn("Champ MemoryBank: failed to load memories:", err);
      }
    }
    this.loaded = true;
  }

  /** Store a memory entry and persist to disk. */
  async store(entry: Omit<MemoryItem, "id" | "timestamp">): Promise<void> {
    const item: MemoryItem = {
      id: `mem-${Date.now().toString(36)}-${Math.random().toString(36).slice(2, 6)}`,
      timestamp: Date.now(),
      ...entry,
    };
    this.items.push(item);
    // Evict oldest entries when over limit.
    if (this.items.length > MAX_MEMORIES) {
      this.items = this.items.slice(this.items.length - MAX_MEMORIES);
    }
    await this.persist();
  }

  /**
   * Return the last `n` memories formatted as a string for system prompt injection.
   * Returns empty string when no memories exist.
   */
  getRecentContext(n = 5): string {
    if (this.items.length === 0) return "";
    const recent = this.items.slice(-n);
    const lines = recent.map(
      (m) =>
        `- User asked: "${m.userQuery}" → "${m.assistantSummary}"`,
    );
    return `## Recent conversation history\n${lines.join("\n")}`;
  }

  private async persist(): Promise<void> {
    try {
      await fs.mkdir(path.dirname(this.filePath), { recursive: true });
      await fs.writeFile(this.filePath, JSON.stringify(this.items, null, 2), "utf-8");
    } catch (err) {
      console.warn("Champ MemoryBank: failed to persist memories:", err);
    }
  }
}
```

In `src/providers/context-manager.ts`, add this method after `fitMessages()`:

```typescript
/**
 * Async variant of fitMessages that summarises dropped turns instead
 * of silently discarding them. Pass a `summarize` callback that accepts
 * an array of dropped messages and returns a plain-text summary.
 * If summarize throws or the summary doesn't help, falls back to the
 * standard drop behaviour.
 */
async fitWithSummary(
  messages: LLMMessage[],
  summarize: (dropped: LLMMessage[]) => Promise<string>,
): Promise<LLMMessage[]> {
  const budget = this.availableTokens(messages);
  if (this.estimateTokens(messages) <= budget) {
    return messages;
  }

  const systemMessages = messages.filter((m) => m.role === "system");
  const rest = messages.filter((m) => m.role !== "system");
  if (rest.length === 0) return messages;

  const lastMessage = rest[rest.length - 1];
  const middle = rest.slice(0, -1);

  // Find how many messages from the front need to be dropped.
  const dropped: LLMMessage[] = [];
  while (middle.length > 0) {
    const candidate = [...systemMessages, ...middle, lastMessage];
    if (this.estimateTokens(candidate) <= budget) break;
    dropped.push(middle.shift()!);
  }

  if (dropped.length === 0) {
    return [...systemMessages, ...middle, lastMessage];
  }

  // Attempt to summarise the dropped turns.
  try {
    const summaryText = await summarize(dropped);
    const summaryMessage: LLMMessage = {
      role: "system",
      content: `[Earlier conversation summary: ${summaryText}]`,
    };
    const withSummary = [...systemMessages, summaryMessage, ...middle, lastMessage];
    // If summary fits, use it; otherwise fall through to plain drop.
    if (this.estimateTokens(withSummary) <= budget) {
      return withSummary;
    }
  } catch {
    // Summarisation failed — fall back to drop.
  }

  // Fallback: plain drop (same as fitMessages).
  return [...systemMessages, ...middle, lastMessage];
}
```

- [ ] **Step 1: Update renderSessionList in webview**

In `webview-ui/dist/main.js`, find `function renderSessionList(sessions, activeSessionId)` and replace the tab construction loop:

```javascript
function renderSessionList(sessions, activeSessionId) {
  lastSessionData = { sessions, activeSessionId };
  tabContainer.innerHTML = '';
  if (!sessions || sessions.length === 0) return;
  for (const s of sessions) {
    const tab = el('div', { class: `tab${s.id === activeSessionId ? ' active' : ''}` });

    // State badge dot (only for non-idle, non-completed states)
    if (s.state === 'running') {
      tab.append(el('span', { class: 'session-dot session-dot-running', title: 'Running' }));
    } else if (s.state === 'errored') {
      tab.append(el('span', { class: 'session-dot session-dot-error', title: 'Error' }));
    } else if (s.state === 'aborted') {
      tab.append(el('span', { class: 'session-dot session-dot-aborted', title: 'Aborted' }));
    }

    const label = el('span', { class: 'tab-label' }, [
      (s.label || 'New chat').slice(0, 22) + ((s.label || '').length > 22 ? '…' : '')
    ]);

    const closeBtn = el('button', { class: 'tab-close', title: 'Close' }, ['×']);
    closeBtn.addEventListener('click', (ev) => {
      ev.stopPropagation();
      vscode.postMessage({ type: 'deleteSessionRequest', sessionId: s.id });
    });

    tab.append(label, closeBtn);
    tab.addEventListener('click', () => {
      if (s.id !== activeSessionId) {
        vscode.postMessage({ type: 'switchSessionRequest', sessionId: s.id });
      }
    });
    tabContainer.append(tab);
  }
}
```
